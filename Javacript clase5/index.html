Conceptos Básicos de JavaScript

1. Variables y Tipos de Datos

En JavaScript existen tres formas principales de declarar variables: var, let y const.

  • var: Tiene scope de función y es hoisted (se eleva su declaración al inicio de la función o script). Sin embargo, puede causar errores difíciles de detectar y ya no se recomienda su uso.
  • let: Tiene scope de bloque (solo existe dentro de las llaves donde se declara). No permite redeclaración en el mismo bloque y es la forma recomendada para variables que cambian su valor.
  • const: También tiene scope de bloque y su valor no puede ser reasignado. Sin embargo, si se usa con objetos o arrays, sus propiedades o elementos sí pueden cambiar. 
let nombre = "Juan"; // Puede cambiar su valor
const edad = 25;     // No puede reasignarse
var esEstudiante = true; // No recomendado

Hoisting es el comportamiento por el cual las declaraciones de variables y funciones se mueven al inicio de su contexto antes de ejecutar el código.

Hoisting en js: Solo var es hoisted completamente (declaración y valor undefined), let y const solo son hoisted en la declaración pero no pueden usarse antes de ser declaradas. 

// Ejemplo de hoisting
console.log(a); // undefined
var a = 5;

// console.log(b); // Error: Cannot access 'b' before initialization
let b = 10;

¿Por qué no usar var? Porque puede provocar errores de scope y hoisting difíciles de depurar. Se recomienda usar let y const.

Tipos de datos

  • Primitivos: string, number, boolean, null, undefined, symbol, bigint
  • No primitivos: object, array, function
let texto = "Hola";      // string
let numero = 42;         // number
let esActivo = false;    // boolean
let nada = null;         // null
let indefinido;          // undefined
let simbolo = Symbol();  // symbol
let persona = { nombre: "Ana" }; // object
let lista = [1,2,3];     // array (object)

const en primitivos: No se puede reasignar el valor.
const en objetos/arrays: No se puede reasignar la variable, pero sí modificar sus propiedades o elementos. 

const x = 5;
// x = 10; // Error

const arr = [1,2,3];
arr.push(4); // Esto sí es válido

2. Operadores

Los operadores permiten realizar operaciones matemáticas, lógicas y de comparación. 

let suma = 5 + 3;           // Operador aritmético
let esMayor = edad > 18;    // Operador de comparación
let esIgual = a === b;      // Comparación estricta
let resultado = cond ? "Sí" : "No"; // Operador ternario

3. Estructuras de Control

if: Permite ejecutar código si se cumple una condición.
if anidado: Un if dentro de otro.
Operador ternario: Forma corta de un if-else. 

if (esEstudiante) {
    console.log("Es estudiante");
} else {
    console.log("No es estudiante");
}

// If anidado
if (edad > 18) {
    if (esEstudiante) {
        console.log("Mayor de edad y estudiante");
    }
}

// Operador ternario
let mensaje = edad >= 18 ? "Mayor de edad" : "Menor de edad";

Booleanos y asignación condicional

let acceso = (edad >= 18); // acceso será true o false según la condición

4. Estructuras repetitivas

Permiten ejecutar un bloque de código varias veces. 

// for clásico
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// while
let contador = 0;
while (contador < 3) {
    console.log(contador);
    contador++;
}

// do...while
let num = 0;
do {
    console.log(num);
    num++;
} while (num < 2);

Switch case

Permite ejecutar diferentes bloques de código según el valor de una variable. 

let dia = 3;
switch (dia) {
    case 1:
        console.log("Lunes");
        break;
    case 2:
        console.log("Martes");
        break;
    case 3:
        console.log("Miércoles");
        break;
    default:
        console.log("Otro día");
}

5. Funciones

Hay varias formas de declarar funciones en JavaScript:

  • Declaración tradicional: 
    function saludar(nombre) {
    return "Hola, " + nombre;
}
  • Función anónima: 
    const sumar = function(a, b) {
    return a + b;
};
  • Función flecha (arrow function): 
    const multiplicar = (a, b) => a * b;

Las funciones flecha no tienen su propio this y son ideales para funciones cortas.

6. Objetos y Arrays

Objetos

Un objeto es una colección de pares clave-valor. 

let persona = {
    nombre: "Luis",
    edad: 30
};
console.log(persona.nombre); // "Luis"

Destructuring

Permite extraer valores de objetos o arrays de forma sencilla. 

const { nombre, edad } = persona;
console.log(nombre); // "Luis"

let numeros = [1, 2, 3];
const [primero, segundo] = numeros;
console.log(primero); // 1

Arrays y métodos básicos

let numeros = [1, 2, 3, 4, 5];
numeros.push(6);    // Agrega al final
numeros.pop();      // Quita el último
numeros.unshift(0); // Agrega al inicio
numeros.shift();    // Quita el primero
let parte = numeros.slice(1, 3); // Extrae una parte (no modifica el original)
numeros.splice(2, 1); // Elimina 1 elemento desde el índice 2
let dobles = numeros.map(n => n * 2); // Crea un nuevo array con los valores multiplicados por 2
let mayores = numeros.filter(n => n > 2); // Nuevo array con los mayores a 2
let suma = numeros.reduce((acc, n) => acc + n, 0); // Suma todos los elementos

Métodos para recorrer arrays

Existen varias formas de recorrer un array y realizar acciones sobre sus elementos:

  • forEach: Ejecuta una función por cada elemento. Recibe como parámetro una función que recibe (elemento, índice, array). 
    let numeros = [1, 2, 3];
numeros.forEach(function(elemento, indice, array) {
    console.log(elemento, indice);
});
  • for...of: Permite recorrer los valores de un array. 
    for (let valor of numeros) {
    console.log(valor);
}
  • map: Crea un nuevo array aplicando una función a cada elemento. Recibe una función con (elemento, índice, array). 
    let dobles = numeros.map(function(elemento) {
    return elemento * 2;
});
console.log(dobles); // [2, 4, 6]
  • filter: Crea un nuevo array con los elementos que cumplen una condición. 
    let mayores = numeros.filter(n => n > 1);
console.log(mayores); // [2, 3]
  • reduce: Reduce el array a un solo valor. Recibe una función con (acumulador, elemento, índice, array) y un valor inicial. 
    let suma = numeros.reduce((acc, n) => acc + n, 0);
console.log(suma); // 6
